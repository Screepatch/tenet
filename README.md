# ImageSplitter - Программа разделения изображений на клетки

Консольная программа на C# для разделения графических файлов на отдельные клетки с возможностью настройки количества строк и столбцов.

## Возможности

- Поддержка форматов: PNG, JPG, JPEG, BMP
- Разделение изображения на заданное количество строк и столбцов
- Автоматическая нумерация сохраняемых клеток
- Обработка остатков при неравномерном делении
- Валидация входных данных и обработка ошибок
- Создание отдельной папки для результатов с временной меткой

## Установка

1. Убедитесь, что у вас установлен .NET 6.0 или новее
2. Склонируйте репозиторий или скачайте исходный код
3. Откройте командную строку в папке с проектом
4. Выполните команду:
   ```
   dotnet build
   ```

## Пример использования

1. Запустите программу:
   ```
   dotnet run
   ```

2. Введите путь к изображению (можно перетащить файл в консоль)
3. Укажите количество строк для разделения
4. Укажите количество столбцов для разделения
5. Программа создаст папку с результатами и сохранит все клетки

### Пример работы:
```
=== Программа разделения изображения на клетки ===

Введите путь к графическому файлу: C:\images\photo.jpg
Изображение загружено: 1200x800 пикселей
Введите количество строк: 3
Введите количество столбцов: 4

Размер каждой клетки: 300x266 пикселей
Обработка...
Обработано клеток: 12/12

Изображение успешно разделено на 12 клеток!
Результаты сохранены в папке: C:\images\photo_split_20250912_143022
```

## Зависимости

- .NET 6.0 или новее
- System.Drawing.Common (автоматически устанавливается через NuGet)

## Структура выходных файлов

Клетки сохраняются в формате PNG с именами:
- `cell_001_row01_col01.png` - первая клетка (верхний левый угол)
- `cell_002_row01_col02.png` - вторая клетка (первая строка, второй столбец)
- и т.д.

## Обработка ошибок

Программа обрабатывает следующие ситуации:
- Несуществующий файл
- Неподдерживаемый формат изображения
- Недостаток памяти
- Некорректные входные данные (отрицательные числа, нечисловые значения)
